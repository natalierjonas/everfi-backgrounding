# **Everfi Backgrounding**

### For this background system, I had Claude help make a dossier analyzing Everfi's whole business history.

## I chose to put the numbers — VC, revenue, etc. — at the top. Then, background info, corporate structure, regulatory and political analyses, and sourcing, so I could best follow the company's evolution. I also thought it would be helpful if I wanted to crossreference choices/changes in the company with historical events or landmark decisions in education.

---

### I mainly wanted to gather info on:
- Lobbying (if any)
- Political relationships
- VC

---

### As I continue to build this workflow, I'd like to implement more about Everfi in the news, building on Enigma's "news" feature.

---

### Limitations or challenges I had

- Effectively combining the findings from three different APIs.
- Getting the proper access points/API keys for these APIs.
- All the APIs I used were free, but I had to sign up to receive my GovInfo API key.
- Enigma was previously downloaded and SBIR doesn't rely on a log-in.

---

### AI use

AI was used to call the APIs, compile the data, organizing the findings chronologically and present the work as a functional, footnoted dossier. To be specific, Claude Code was used.
